Konveyor AI Geliştirici Kılavuzu

Endüstriyel üretim hatları için tasarlanmış, yapay zeka destekli konveyör bandı güvenlik sisteminin teknik mimarisi, kurulumu ve çalışma prensipleri.

Kontrol Paneline Git

Sistem Mimarisi ve Çalışma Algoritması

1

Otomatik Referanslama (Kalibrasyon)

Sistem ilk kez başlatıldığında veya kullanıcı tarafından tetiklendiğinde, canlı video akışından konveyör bandının mevcut ideal konumunu analiz eder ve bunu bir "başlangıç referansı" olarak kaydeder.

2

Gerçek Zamanlı AI Analizi

Google Gemini modeliyle güçlendirilmiş Genkit akışı, video akışından periyodik olarak (her 2 saniyede bir) görüntü kareleri alır. Bu kareleri, başlangıç referansıyla karşılaştırarak bandın kenarındaki olası kaymaları milimetre cinsinden hesaplar.

3

Anomali Tespiti ve Görselleştirme

Hesaplanan sapma değeri, kullanıcı tarafından belirlenen "anomali eşiğini" aşarsa, sistem durumu "ANOMALİ" olarak değiştirilir, görsel/sesli uyarılar verilir ve bu olay kayıt defterine eklenir. Tüm veriler anlık olarak arayüzde görselleştirilir.

Teknoloji Yığını

  • Next.js (App Router)
  • TypeScript
  • Google Genkit (Gemini)
  • Shadcn UI
  • Tailwind CSS
  • Lucide React

Kurulum ve Başlatma

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin.

1. Bağımlılıkları Yükleyin:

npm install

2. Geliştirme Sunucusunu Başlatın:

npm run dev

Uygulama varsayılan olarak http://localhost:9002 adresinde çalışmaya başlayacaktır.
