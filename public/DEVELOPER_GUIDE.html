Teknik Gereksinimler

Projeyi yerel makinenizde sorunsuz bir şekilde çalıştırmak için aşağıdaki araçların kurulu olması gerekmektedir:

  • Node.js (v18.x veya üzeri)
  • npm (v9.x veya üzeri)
  • Modern bir web tarayıcısı (Chrome, Firefox, Edge)

Kurulum ve Başlatma

Projeyi başlatmak için aşağıdaki komutları terminalde sırasıyla çalıştırın:

  1. Bağımlılıkları Yükle: npm install
  2. Geliştirme Sunucusunu Başlat: npm run dev

Uygulama varsayılan olarak http://localhost:9002 adresinde çalışacaktır.

Proje Yapısı ve Hiyerarşi

Proje, Next.js App Router mimarisine uygun olarak modüler bir yapıda tasarlanmıştır:

  • src/app/: Ana sayfa ve panel yönlendirmeleri.
  • src/components/: Tekrar kullanılabilir UI bileşenleri.
  • src/ai/: Google Genkit ile yapay zeka akışları.
  • public/: Statik varlıklar (video, ses dosyaları).

Temel Teknolojiler

Konveyor AI, modern ve güçlü teknolojiler kullanılarak geliştirilmiştir:

  • Next.js: Sunucu tarafı render ve performans optimizasyonu.
  • Genkit (Gemini): Görüntü analizi için yapay zeka altyapısı.
  • Shadcn UI & Tailwind: Hızlı ve estetik arayüz geliştirme.
  • TypeScript: Kod kalitesi ve güvenliği.
