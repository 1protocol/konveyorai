Teknik Gereksinimler

  • Node.js 20.x veya üstü
  • npm 10.x veya üstü
  • Modern bir web tarayıcısı (Chrome, Firefox, Edge)
  • Google Gemini API anahtarı (.env dosyasında)

Kurulum ve Başlatma

Projeyi yerel makinenizde çalıştırmak için:

# 1. Bağımlılıkları yükleyin
npm install

# 2. Geliştirme sunucusunu başlatın
npm run dev

Uygulama `http://localhost:9002` adresinde açılacaktır.

Proje Yapısı ve Hiyerarşi

Proje, modern Next.js App Router yapısını takip eder:

  • `src/app`: Ana sayfalar ve routing.
  • `src/components`: Tekrar kullanılabilir React bileşenleri.
  • `src/ai`: Google Genkit ile ilgili tüm yapay zeka akışları.
  • `src/hooks`: Özel React hook'ları.
  • `public`: Videolar, sesler gibi statik varlıklar.

Temel Teknolojiler

  • Next.js: Sunucu ve istemci tarafı render için.
  • TypeScript: Tip güvenliği ve ölçeklenebilirlik.
  • Genkit (Gemini): Görüntü analizi için yapay zeka modeli.
  • Shadcn UI: Önceden oluşturulmuş, esnek UI bileşenleri.
  • Tailwind CSS: Hızlı ve modern arayüz tasarımı.
