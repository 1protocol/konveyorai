Teknik Gereksinimler

Projeyi yerel makinenizde sorunsuz bir şekilde çalıştırmak için aşağıdaki araçların kurulu olduğundan emin olun:

  • Node.js (v18.x veya üzeri)
  • npm (Genellikle Node.js ile birlikte gelir)
  • Modern bir web tarayıcısı (Chrome, Firefox, Edge)

Kurulum ve Başlatma

Projeyi başlatmak için terminalde aşağıdaki adımları izleyin:

# 1. Proje bağımlılıklarını yükleyin
npm install

# 2. Geliştirme sunucusunu başlatın
npm run dev

Uygulama http://localhost:9002 adresinde çalışmaya başlayacaktır.

Temel Teknolojiler

Proje, modern ve performans odaklı teknolojiler üzerine kurulmuştur:

  • Next.js: Sunucu tarafı render ve statik site oluşturma yetenekleriyle güçlü bir React framework'ü.
  • Genkit (Gemini): Görüntü analizi ve anomali tespiti için kullanılan yapay zeka altyapısı.
  • Shadcn UI: Tekrar kullanılabilir ve özelleştirilebilir modern UI bileşenleri sağlar.
  • Tailwind CSS: Hızlı ve pratik bir şekilde stil oluşturmayı sağlayan bir CSS framework'ü.

Proje Yapısı ve Hiyerarşi

Projenin ana dizin yapısı, modülerliği ve okunabilirliği artırmak için özenle tasarlanmıştır:

  • /src/app: Uygulamanın ana sayfaları ve yönlendirme mantığı.
  • /src/components: Arayüzde kullanılan tekrar edilebilir React bileşenleri.
  • /src/ai: Genkit akışları ve yapay zeka ile ilgili tüm kodlar.
  • /public: Videolar, resimler ve ses dosyaları gibi statik varlıklar.

Tasarım Felsefesi ve Teknik Detaylar: "VR Cam Çerçeve" Efekti

Arayüzde kullanılan modern "VR Cam Çerçeve" kart efektinin arkasındaki HTML ve CSS yapısı aşağıda detaylandırılmıştır. Bu efekt, NFT kartlarından ve fütüristik kullanıcı arayüzlerinden ilham alınarak tasarlanmıştır.

1. Katmanlı HTML Yapısı

Efektin temelini, iç içe geçmiş iki katmanlı bir HTML yapısı oluşturur. Bu yapı, derinlik ve görsel ayrım kazandırma esnekliği sunar.

<div class="card"> <!-- Dış Katman (Hareketli Çerçeve) -->
  <div class="card-inner"> <!-- İç Katman (Cam Panel) -->
    <!-- İçerik buraya gelir -->
  </div>
</div>

2. CSS ile Hareketli "VR" Kenarlık

Kartın en dikkat çekici kısmı, etrafında dönen çok renkli ışıktır. Bu, CSS'in conic-gradient ve @keyframes özellikleri kullanılarak elde edilir.

.card::after {
  content: '';
  position: absolute;
  top: -50%; left: -50%;
  width: 200%; height: 200%;
  background: conic-gradient(...);
  animation: border-glow 8s linear infinite;
}

@keyframes border-glow {
  100% { transform: rotate(360deg); }
}

3. "Glassmorphism" İç Panel

İçerik paneli, hareketli çerçevenin üzerinde yüzen bir cam parçası hissi verir. Bu, background: rgba(...), backdrop-filter: blur(...) ve z-index özelliklerinin birleşimiyle sağlanır.
