Konveyor AI Geliştirici Kılavuzu

Endüstriyel üretim hatları için tasarlanmış, yapay zeka destekli konveyör bandı güvenlik sisteminin teknik mimarisine, algoritmalarına ve kurulum adımlarına hoş geldiniz.

Operasyon Paneline Git

Sistem Mimarisi ve Çalışma Algoritması

Konveyor AI, gerçek zamanlı görüntü işleme ve yapay zeka analizi üzerine kurulu üç temel aşamadan oluşan bir mimariyle çalışır:

Adım 1: Otomatik Referanslama

Sistem, kalibrasyon komutuyla konveyör bandının o anki görüntüsünü analiz eder ve bunu "ideal" veya "referans" konum olarak kaydeder. Bu referans, sapma hesaplamaları için temel ölçüt olur.

Adım 2: Gerçek Zamanlı AI Analizi

Canlı video akışından periyodik olarak alınan kareler (frame), Google Genkit (Gemini) üzerinde çalışan AI modeline gönderilir. Model, görüntüyü referansla karşılaştırır ve olası dikey kaymayı milimetre cinsinden hesaplar.

Adım 3: Anomali Tespiti ve Uyarı

AI'dan dönen sapma değeri, kullanıcı tarafından belirlenen "anomali eşiği" ile karşılaştırılır. Eğer sapma eşikten büyükse, sistem "ANOMALİ" durumuna geçer, görsel/sesli uyarılar verir ve olayı kayıt defterine işler.

Teknoloji Yığını

  • Next.js (App Router)
  • TypeScript
  • Google Genkit (Gemini)
  • Shadcn UI
  • Tailwind CSS

Proje Yapısı

Proje, modüler ve ölçeklenebilir bir yapı sağlamak için Next.js App Router standartlarına göre düzenlenmiştir. Ana dizinler ve görevleri aşağıda özetlenmiştir:

.
├── public/                 # Statik varlıklar (video, ses, HTML kılavuzu)
│   ├── conveyor-video.mp4
│   ├── alert-sound.mp3
│   └── DEVELOPER_GUIDE.html
│
├── src/
│   ├── app/                # Sayfalar, layout'lar ve rota bileşenleri
│   │   ├── dashboard/
│   │   │   └── page.tsx    # Ana kontrol paneli sayfası
│   │   ├── layout.tsx      # Ana layout
│   │   └── page.tsx        # Geliştirici kılavuzunu gösteren giriş sayfası
│   │
│   ├── components/         # Tekrar kullanılabilir React bileşenleri
│   │   ├── ui/             # Shadcn UI temel bileşenleri
│   │   └── dashboard-client.tsx # Panelin tüm istemci tarafı mantığı
│   │
│   ├── ai/                 # Genkit ve yapay zeka ile ilgili kodlar
│   │   ├── flows/
│   │   │   └── analyze-conveyor-flow.ts # Görüntü analizi yapan AI akışı
│   │
│   ├── hooks/              # Özel React hook'ları (use-toast, use-mobile)
│   │
│   └── lib/                # Yardımcı fonksiyonlar ve TypeScript tanımları
│
└── package.json            # Proje bağımlılıkları ve script'ler

Kurulum ve Başlatma

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin.

1. Bağımlılıkları Yükleyin:

npm install

2. Geliştirme Sunucusunu Başlatın:

npm run dev

Uygulama varsayılan olarak http://localhost:9002 adresinde çalışmaya başlayacaktır. Geliştirici Kılavuzu ana sayfada, operasyon paneli ise /dashboard altında yer alır.
