Geliştirme Sorumlusu

Teknik Gereksinimler

Projeyi yerel makinenizde sorunsuz bir şekilde çalıştırmak için aşağıdaki araçların yüklü olduğundan emin olun:

  • Node.js: v20.11.1 veya üzeri
  • npm: v10.2.4 veya üzeri
  • Tarayıcı: Google Chrome, Firefox, veya Edge'in güncel sürümleri.

Kurulum ve Başlatma

Projeyi başlatmak için aşağıdaki komutları terminalinizde çalıştırın:

# 1. Proje bağımlılıklarını yükleyin
npm install

# 2. Geliştirme sunucusunu başlatın
npm run dev

Uygulama varsayılan olarak http://localhost:9002 adresinde çalışacaktır.

Proje Yapısı ve Hiyerarşi

Proje, modern Next.js (App Router) standartlarına göre yapılandırılmıştır. Önemli dizinler ve görevleri:

.
├── src/
│   ├── app/                # Ana sayfa ve dashboard yönlendirmeleri
│   ├── components/         # Tekrar kullanılabilir UI bileşenleri (Shadcn)
│   ├── ai/                 # Google Genkit yapay zeka akışları
│   └── public/             # Statik dosyalar (video, ses, bu kılavuz)
└── package.json            # Proje bağımlılıkları

Temel Teknolojiler

Bu proje, endüstri standardı ve modern teknolojiler kullanılarak geliştirilmiştir:

  • Next.js: React tabanlı, sunucu tarafı render yetenekleri sunan framework.
  • TypeScript: Kod kalitesini ve güvenliğini artıran statik tip denetleyicisi.
  • Google Genkit: Görüntü analizi için Gemini modelini kullanan yapay zeka altyapısı.
  • Shadcn UI: Kopyala-yapıştır mimarisine sahip, özelleştirilebilir bileşen kütüphanesi.
  • Tailwind CSS: Hızlı arayüz geliştirmeyi sağlayan "utility-first" CSS çatısı.
