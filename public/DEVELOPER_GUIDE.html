Konveyor AI Geliştirici Kılavuzu

Yapay zeka destekli konveyör güvenliğinde devrim yaratın.

Operasyon Paneline Git

Teknik Gereksinimler

  • Node.js v18.x veya üstü
  • npm veya uyumlu bir paket yöneticisi
  • Modern bir web tarayıcısı (Chrome, Firefox, Edge önerilir)
  • Görüntü işleme için donanım hızlandırma desteği

Kurulum ve Başlatma

Projeyi yerel makinenizde hızla ayağa kaldırın.

  1. Bağımlılıkları yükleyin: npm install
  2. Geliştirme sunucusunu başlatın: npm run dev
  3. Uygulamaya http://localhost:9002 adresinden erişin.

Temel Teknolojiler

  • Next.js: Sunucu taraflı render ve modern React özellikleri için.
  • TypeScript: Kod kalitesi ve tip güvenliği için.
  • Genkit (Gemini): Görüntü analizi yapan yapay zeka modeli için.
  • Shadcn UI: Tekrar kullanılabilir ve estetik arayüz bileşenleri için.
  • Tailwind CSS: Hızlı ve esnek stil yönetimi için.

Proje Yapısı ve Hiyerarşi

Temel klasörler ve görevleri:

  • /src/app: Uygulamanın ana sayfaları ve yönlendirme mantığı.
  • /src/components: Tekrar kullanılabilir arayüz bileşenleri.
  • /src/ai: Genkit akışları ve yapay zeka yapılandırması.
  • /public: Videolar, sesler gibi statik varlıklar.
